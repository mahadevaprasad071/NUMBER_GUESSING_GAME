# 🎯 Number Guessing Game

A simple Python-based Number Guessing Game where the computer generates a random number between 1 and 100, and the player tries to guess it.

## 📌 About The Project

This game uses Python's random module to generate a secret number. The player enters guesses, and the program gives hints like "Too low" or "Too high" until the correct number is found.

## 💻 Technologies Used

- Python 🐍
- Random Module

## 🎮 How To Play

1. Run the Python program.
2. Enter a number between 1 and 100.
3. Follow the hints:
   - Too low → Guess a higher number
   - Too high → Guess a lower number
4. Guess the correct number and win 🎉
